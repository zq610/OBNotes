## 1、定义
链表：由指针连接的结构体。
```C++
struct Node{
	int data;
	Node* next;
};
```
补充：存储期[[存储期（分配内存~释放内存）]]
## 2、操作
### 1、创建
```c++
Node* creatlist(int n)
{
	// Node* head{new Node{0}};
	Node* head  = new Node;
	head->next = NULL;
	Node* pre = head;
	for(int i = 0;i<n;i++)
	{
		Node* current = new Node;
		current->data = i+1;
		pre->next = current;
		pre = current;
		current->next = NULL;
	}
	return head;
}
```
### 2、遍历
```C++
void ShowLinkList(Node* head)
{
	Node* current = head->next;
	int e;
	while(current!=NULL)
	{
		e = current->data;
		cout<<e<<" ";
		current = current->next;
	}
}
```
