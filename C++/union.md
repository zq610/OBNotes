# 1、定义

所有数据成员共t用一个内存块的特殊的类。

# 2、特点

- 默认访问控制public

- 可以包含构造、析构函数

- 不能还有引用类型成员

- 不能继承自其他类，不能做基类

- 不能含有虚函数

- 匿名共用体union在作用域内可直接访问union成员

- 匿名union不能有private和protected成员

- 全局匿名union必须是静态的

# 3、用途

## 1、用于类型双关(类型转化)

```cpp

union

{

    float a;

    int b;

}

//a,b共用内存，值相互影响，可完成类型转换
```
