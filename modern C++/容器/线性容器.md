# std::forward_list

列表容器，与std::list相似

1、std::forward_list

- 是单向链表实现，不支持快速随机访问，但插入复杂度为o（1）

- STL中唯一不提供size()方法的容器

- 不需要双向迭代时，使用该list空间利用率更高

2、std::list

- 由双向链表实现

- 适用于双向迭代
